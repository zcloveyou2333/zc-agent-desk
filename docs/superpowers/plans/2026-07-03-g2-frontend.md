# G2 Frontend Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver the selected three-column React interface for the persisted deterministic Mock Runtime.

**Architecture:** Vite serves a single React application. `api.ts` owns HTTP/SSE transport, `types.ts` owns wire contracts, and focused components render navigation, chat/approval, Trace, and todos. FastAPI remains authoritative for run state and side effects.

**Tech Stack:** React 19, TypeScript, Vite, Vitest, Testing Library, native EventSource-compatible SSE parser, CSS.
