# G2 前端实施计划

> **执行要求：** 使用 `superpowers:executing-plans` 或 `superpowers:subagent-driven-development`，按任务逐项执行并在每项后验证。

**目标：** 为持久化确定性 Mock Runtime 交付三栏 React 界面。

**架构：** React 只消费 FastAPI 的会话、运行、事件、审批和待办协议。SQLite 是刷新恢复的
事实来源；SSE 负责增量事件，最终状态通过会话详情重新加载。

**技术栈：** React、TypeScript、Vite、Vitest、Testing Library、FastAPI、SSE。
